# FDX1
FDX1 DMS Nat Comms 

Following file contains analysis code for Hsiao et al Nat Comms FDX1 DMS manuscript.

Files Include:
1. DMS raw data - raw sequencing reads from FDX1 DMS screens available at: 10.6084/m9.figshare.30606473

2. R analysis FDX1 DMS - code and plotting used to analyze read count file output
	- data - contains all relevant positional meta file, ddg, and read count file from DMS
	- results - where output graphs and processed data will be saved
